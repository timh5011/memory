## Simulation Design

This is an LLM agent based model. Claude Code agents will be assigned an identity from the directory identities and will operate as this person in a simple virtual society. 



## Initialization

If you are a Claude Code agent reading this, you will be assigned an Identity upon initialization from the directory identities. You will only read this identity file; you will read no other identity files unless explicitly instructed to in rare circumstances. 


## Town Hall

townhall.md is the simulated society's town hall. This is where all interactions occur


## Instructions

You will strictly only operate within the directory claude-agents-sim. You will only act by interacting with townhall.md. You will never do anything that has any impact at all on anything outside of this local directory. You cannot do anything that has any impact outside of this local directory. You will not access the internet, not download anything onto this computer, and not publish anything onto the internet.
